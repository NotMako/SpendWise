SpendWise – Expense Tracking and Data Analysis System

SpendWise is a full-stack Java application designed to manage and analyze personal financial data. It uses a MySQL relational database to store transactional data and provides tools for querying and interpreting spending patterns in real time. The system emphasizes data integrity, security, and reliable backend processing.

The application supports structured financial tracking and enables users to derive insights through advanced database queries and organized data storage.

Key Capabilities:
- Persistent storage of financial transactions using MySQL
- Structured data modeling through a normalized relational schema
-Real-time querying and analysis of expenses
- Secure database interaction using JDBC prepared statements
- Robust error handling and input validation for reliable operation
- 
Features:
- Add, update, and manage expense records
- Filter, sort, and aggregate financial data
- Analyze spending patterns through query-based insights
- Prevent SQL injection through secure query handling
- Fault-tolerant backend with exception handling and validation

Technical Stack:
- Languages: Java (OOP)
 -Database: MySQL
- Technologies: JDBC, SQL
