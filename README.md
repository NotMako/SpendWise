# SpendWise
SpendWise is a full-stack Java expense tracking application designed for efficient and secure financial data management.

The system uses a MySQL relational database with a normalized schema to ensure data integrity and support scalable storage of transactional data. It features advanced querying capabilities including sorting, filtering, and aggregation, enabling users to analyze expenses in real time.

Robust backend design principles were applied, including JDBC prepared statements to prevent SQL injection, comprehensive input validation, and structured exception handling to ensure reliability and fault tolerance.

This project demonstrates experience in backend development, database design, and building data-driven applications using Java and SQL.
